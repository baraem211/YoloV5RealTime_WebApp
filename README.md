# 🎯 YOLOv5 실시간 객체 탐지 웹앱

브라우저에서 바로 실행되는 실시간 객체 탐지 웹 애플리케이션입니다.  
**TensorFlow.js + COCO-SSD(MobileNet V2)** 모델을 사용하여 웹캠 스트림에서 실시간으로 80가지 객체를 탐지합니다.

> 📌 참고: [EmmettHwang/yolov5ReadTime](https://github.com/EmmettHwang/yolov5ReadTime) - Python 기반 YOLOv5 실시간 탐지 레포지토리

## 🎮 사용 방법

1. 웹앱 접속 시 **COCO-SSD 모델 자동 로드** (최초 1회, 약 3~5초 소요)
2. **"탐지 시작"** 버튼 클릭 → 브라우저 카메라 권한 허용
3. 실시간으로 객체 탐지 시작
4. 설정 패널에서 **신뢰도 임계값 / 프레임 스킵 / 박스 스타일** 조절
5. **스냅샷** 버튼으로 현재 탐지 결과 이미지 저장

## ⚠️ 미구현 기능 / 제한사항

- **실제 YOLOv5 모델 사용 불가**: 브라우저 환경에서 PyTorch YOLOv5 직접 실행 불가 → COCO-SSD로 대체
- **비디오 파일 입력**: 현재 웹캠만 지원 (파일 업로드 탐지 미구현)
- **후면 카메라 선택**: 모바일 후면 카메라 전환 기능 미구현
- **결과 CSV 내보내기**: 탐지 로그 파일 저장 미구현
